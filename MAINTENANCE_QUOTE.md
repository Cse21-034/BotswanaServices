## 1. PURPOSE OF THIS DOCUMENT

This document constitutes a formal quotation and proposed agreement for the ongoing maintenance, support, and management of a custom-built web application developed by the Service Provider. It outlines the scope of services, responsibilities, deliverables, pricing, and terms applicable to the maintenance engagement.

Upon acceptance by both parties and collection of the first payment, this document shall serve as a binding Software Maintenance & Support Agreement governed by the laws of the Republic of Botswana.

---

## 2. DESCRIPTION OF THE SOFTWARE SYSTEM

The system subject to this agreement is a **full-stack web application** comprising the following technical components:

| Component | Technology / Platform |
|---|---|
| Frontend Framework | Next.js 14 (React) |
| Backend / API | Next.js API Routes (Node.js) |
| Database | PostgreSQL via Prisma ORM |
| Cloud Storage | Supabase Storage / Cloudinary |
| Authentication | NextAuth.js |
| Email Service | Resend (Transactional Emails) |
| Payment Gateway | PayGate (Card Payments) |
| Deployment Platform | Vercel |
| Version Control | GitHub |

The system includes, but is not limited to, the following functional modules:

- Business listing directory with search and filtering
- Multi-tier subscription and billing system (monthly and yearly)
- Admin panel for business approval and management
- Bookings and appointment management
- In-app notification system
- Email notification system (welcome, payment, review, booking, status)
- Business analytics dashboard
- Photo and video upload for business profiles
- Membership and accreditation management
- Review and rating system
- WhatsApp integration for customer contact

## 3. SCOPE OF MAINTENANCE SERVICES

The following services are included under this maintenance agreement, subject to the selected package (Section 5):

### 3.1 Corrective Maintenance
- Investigation and resolution of software bugs, errors, and defects
- Fixing broken API integrations (PayGate, Supabase, Cloudinary, Resend, etc.)
- Database error resolution and data integrity checks
- Resolving server-side exceptions and runtime errors

### 3.2 Security Maintenance
- Regular updates to dependencies and packages to address known vulnerabilities
- Monitoring for security advisories affecting the technology stack
- Implementation of security patches as required
- SSL certificate renewal oversight
- Review of authentication and access control configurations

### 3.3 Preventive Maintenance
- Routine review and optimisation of database queries
- Monitoring of application performance and uptime
- Regular code health reviews
- Proactive identification of potential failure points

### 3.4 Infrastructure & Platform Maintenance
- Management of Vercel deployment configurations
- Supabase database and storage monitoring
- Cloudinary media library management
- Environment variable and secrets management
- Domain and DNS oversight (advisory only)

### 3.5 Minor Enhancements (Standard & Premium Only)
- Text, label, and copy changes
- Colour, branding, and layout adjustments
- Addition of new form fields to existing forms
- Adjustment of subscription plan pricing or features
- Configuration of new email notification templates

### 3.6 Technical Support
- Dedicated communication channel (WhatsApp / Email)
- Response to technical queries from the client
- Guidance on system usage and administration
- Monthly status and activity report (Premium only)

## 4. EXCLUSIONS — WHAT IS NOT COVERED

The following are **not** included under any maintenance package and will be quoted separately:

- Development of entirely new features or modules
- Redesign of the user interface or user experience
- Migration to a different technology stack or platform
- Integration of new third-party payment gateways or APIs
- Mobile application (Android / iOS) development
- Data entry, content population, or SEO services
- Third-party vendor fees (Vercel, Supabase, Cloudinary, Resend, PayGate)
- Domain registration or hosting costs
- Hardware or device support

Any work outside this scope will be subject to a separate quotation at a rate of **BWP 450.00 per hour**.

## 6. PAYMENT TERMS

1. **Invoicing:** Invoices will be issued on the **1st of each calendar month** for the upcoming month's service.
2. **Due Date:** Payment is due within **7 calendar days** of the invoice date.
3. **Payment Method:** Electronic Funds Transfer (EFT) to banking details provided on invoice.
4. **Late Payment:** A penalty of **1.5% per month** will be applied to overdue balances.
5. **Annual Billing:** Clients opting for annual billing receive the discount stated above and payment is due in full before commencement of the annual term.
6. **Currency:** All payments must be made in **Botswana Pula (BWP)**.

## 7. CONTRACT DURATION & TERMINATION

1. This agreement commences on the date the first invoice is settled.
2. Monthly packages continue on a **rolling month-to-month** basis.
3. **Cancellation Notice:** Either party may terminate this agreement by providing **30 days written notice** via email.
4. No refunds will be issued for partial months if the client cancels mid-month.
5. Upon termination, the Service Provider will provide the client with all source code, database exports, and credentials within **14 business days**.
6. The Service Provider reserves the right to suspend services immediately in the event of non-payment exceeding **14 days** beyond the due date.
